Audio output

If dialogue sometimes goes silent while music and effects keep playing, choose Stereo — TVs have two speakers, so the server folds surround (5.1) down to stereo correctly and keeps dialogue out front. Surround sends the original track untouched, for a real surround system.

Dialogue boost

How far to push spoken dialogue forward while folding surround down. Strong is best if voices are still hard to hear. Only affects surround (5.1) sources played in Stereo.

Night mode

Evens out loud and quiet parts so explosions don't blow out dialogue at low volume — good for late-night watching.

Loudness normalization

Keeps the overall volume consistent from one title to the next, so you're not reaching for the remote between films.
